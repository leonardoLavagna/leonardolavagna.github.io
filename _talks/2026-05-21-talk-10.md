Presentation for a [journal club](https://en.wikipedia.org/wiki/Journal_club) session at [CERN](https://home.cern/). I discussed the implementation details about an orchestration framework I have developed to
run some specific quantum-enhanced tasks (e.g., circuit cutting and sample-based-diagonalization) on a cluster using [Kubernetes](https://kubernetes.io/). The framework works also 
with a common user interface that scientists without previous experience on cluster computing can use. The slides are available [here](/assets/files/cern_jc-2026.pdf). If possible I will also upload the code as soon as
the beta-testing phase is completed.
